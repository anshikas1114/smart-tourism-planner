Demo Flow – Smart Tourism Planner
Theme: Virasat se Vikas Tak
1. Purpose of the Demo

The purpose of this demo is to showcase how modern web technology can digitally enhance and promote Indian heritage tourism. The demo focuses on a simplified but impactful user journey that highlights cultural exploration, smart route planning, and budget-based trip organization within a single platform.

2. Demo Scope

This demo version is a prototype, not a full production system.
Included in Demo-
  City-wise heritage exploration
  Predefined cultural data (heritage sites, food, festivals)
  Map-based route visualization
  Budget and day-based trip planning
Out of Scope-
  User authentication
  Real-time bookings
  Payments
  Admin dashboard

3. Step-by-Step Demo Flow-

Step 1: Home Page
User lands on the homepage.
Brief introduction to the project theme and purpose.
Call-to-action button: “Plan Your Trip”.

Step 2: City Selection
User selects a city (e.g., Jaipur).
The system loads predefined cultural and heritage data for the selected city.

Step 3: Trip Preferences
User selects:
Number of days
Budget range (Low / Medium / High)
The system filters relevant places based on selected preferences.

Step 4: Smart Trip Planning
The system generates:
A day-wise itinerary
Recommended heritage sites
Local food and cultural highlights
Logic is based on predefined rules and distance optimization.

Step 5: Map Visualization
Optimized route is displayed using Google Maps API.
Markers show heritage locations.
Total distance and route path are visualized.

Step 6: Cultural Context
User views additional cultural information:
Local foods
Festivals
Traditional markets (if applicable)
This reinforces the Virasat aspect of the project.

5. Key Highlights for Judges

City-based tourism planning
Integration of cultural heritage with modern mapping technology
Simple, intuitive user experience
Clear connection between tradition and technological development

6. Demo Conclusion

The demo concludes by emphasizing how the Smart Tourism Planner transforms traditional tourism information into a structured, intelligent, and user-friendly digital solution, aligning perfectly with the theme “Virasat se Vikas Tak.”

7. Future Scope (Brief Mention During Demo)
